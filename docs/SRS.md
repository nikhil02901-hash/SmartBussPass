# Software Requirements Specification (SRS)

## Overview
Smart Bus Pass Management System with QR verification and fraud risk scoring.

## Modules
- Authentication
- Apply Pass (Wizard)
- Admin Dashboard (Search/Filter/Pagination)
- Fraud Scoring (ML-ready)
- QR Verification
- PDF Pass Generation
- Renewals
- Audit Logs

## Non-functional
Security, logging, usability, performance.
